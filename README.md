# Maze Generator and Path Validator
This is a console-based C++ program that generates a random maze using a 2D array.  
The maze contains walls (`1`), paths (`0`), a start point (`S`), an end point (`E`), and the player's current position (`P`).

## Features
- User-defined maze size from 5x5 to 20x20
- Random maze generation with a guaranteed valid path
- Additional fake paths and dead ends
- WASD movement input
- Move validation to prevent hitting walls or leaving the maze
- Current position display
- Path history and move counter
- Menu-based interaction

## Technologies Used
- C++
- 2D arrays
- Functions
- Input validation
- Random number generation

## How to Run
Compile and run the program using a C++ compiler such as g++ or Visual Studio.
